# Sistema de Ventas - Lógica de Programación

Este proyecto corresponde a la materia Lógica de Programación.
El objetivo del sistema es registrar productos, consultar los registros
y finalizar una venta mostrando un resumen final.

## Funcionalidades
- Registro de uno o varios productos
- Validación del precio ingresado
- Consulta de productos registrados
- Cálculo del total a pagar
- Resumen final de la venta

## Tecnologías utilizadas
- Lenguaje de programación: Python
- Entorno de desarrollo: Visual Studio Code
- Control de versiones: GitHub

## Estructuras utilizadas
- Variables y tipos de datos
- Condicionales (if, else)
- Estructuras repetitivas (while)
- Listas para almacenar datos

## Diagramas de flujo
El proyecto incluye diagramas de flujo que representan:
- El funcionamiento general del sistema
- El proceso de registro de productos
